# React, TS, TW, Vite, Vitest Scaffold

Scaffold / template for creating a new React, Typescript, Tailwind, Vite and Vitest app.

## Init from scratch

npx degit crunchwrap89/React-TS-TW-Vite-Vitest-scaffold

## Install

```bash
npm install
```

## Run Dev

```bash
npm run dev
```

## Run Tests

```bash
npm run test
```

## Lint

```bash
npm run lint
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
